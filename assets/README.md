# Assets

Static assets for Sona AI OS including images, icons, fonts, and media files.

## Structure

```
assets/
├── icons/     — Application icons and favicons
├── images/    — Screenshots, diagrams, and illustrations
├── fonts/     — Custom font files
└── media/     — Audio, video, and other media assets
```

## Purpose

Central repository for all static assets used across platforms:

- Application branding (logos, icons)
- Documentation images and diagrams
- UI assets shared between frontend platforms
- Marketing and promotional materials

## Guidelines

- Use SVG for scalable graphics where possible
- Optimize raster images before committing
- Include @1x, @2x, @3x variants for mobile
- Document the purpose and usage of each asset
- Respect licensing for any third-party assets
