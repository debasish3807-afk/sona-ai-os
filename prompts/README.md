# Prompts

Curated prompt library for Sona AI OS agents and capabilities.

## Structure

```
prompts/
├── coding/      — Code generation, review, and debugging prompts
├── research/    — Research, analysis, and summarization prompts
├── automation/  — Task automation and workflow prompts
├── agents/      — Agent-specific behavioral prompts
└── system/      — Core system prompts and meta-prompts
```

## Categories

### Coding (`coding/`)
Prompts for software development tasks: code generation, refactoring, debugging, documentation, and code review.

### Research (`research/`)
Prompts for information gathering, analysis, summarization, and knowledge synthesis.

### Automation (`automation/`)
Prompts for workflow automation, task decomposition, and process optimization.

### Agents (`agents/`)
Behavioral prompts that define how specialized agents think, reason, and respond.

### System (`system/`)
Core system prompts including base personality, safety guidelines, and meta-prompting templates.

## Guidelines

- Write clear, specific prompts with examples where helpful
- Include input/output format specifications
- Document expected model and parameters
- Test prompts across multiple scenarios before committing
- Use template variables with `{{variable_name}}` syntax
